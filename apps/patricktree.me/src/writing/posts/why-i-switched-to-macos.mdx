- bad things about Windows
  - short freezings after hibernating and resuming
  - File Explorer is complete trash (bad navigation via keyboard, bad previewing of files)
  - itself not great for development (git bash is OK but everything is slow and native javascript dependencies might not work properly)
  - Subsystem for Linux combined with VS Code is really OK for development (see also my tidbit `wsl-and-ubuntu-web-dev-setup`) but 
- great thing about MacOS / MacBook Pro
  - fantastic battery runtime
  - great performance
  - Finder is actually good
  - three modifier keys (Control, Option and Command) make shortcuts in general shorter (Windows has really only two modifier keys, Control and Alt - "Windows" key is almost never used by applications for shortcuts)
  - brew for package management
  - Ghostty is a great terminal
